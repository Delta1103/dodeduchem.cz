Rezervační systém - Den otevřených dveří (ZŠ)

Jednoduchá lokální aplikace (Chrome). Data jsou uložena v localStorage pro daný prohlížeč. Administrátor přidává termíny, učitelky se na ně přihlašují. Každý termín lze obsadit pouze jednou.

Přidat / importovat termíny (admin)

Každý termín obsahuje: datum (YYYY-MM-DD), start (HH:MM), místo (Meziboří|Most). Délka je přednastavená podle místa (Meziboří = 90 min, Most = 60 min).

Dokumentace (stručně)

1) Účel - Systém umožňuje adminovi (škola) vytvářet termíny DOD (datum, start, místo). - Učitelky se mohou přihlásit na konkrétní termín; každý termín může být obsazen jen jednou. 2) Délky prohlídek - Meziboří = 90 minut - Most = 60 minut - Po rezervaci se uloží místo a start; konec lze dopočítat: konec = start + délka. 3) Jak používat (Chrome) - Otevřete tento HTML soubor v Chrome (File -> Open). - Admin přidá termíny ručně nebo importem CSV (formát: YYYY-MM-DD,HH:MM,Meziboří|Most). - Učitelka vyplní jméno, email, vybere volný termín a klikne Rezervovat. - Systém vygeneruje krátký kód pro zrušení rezervace (zobrazen v tabulce). Uložit. - Zrušení: vložte kód a klikněte Zrušit. 4) Uložení dat - Data jsou lokálně v localStorage pro tento prohlížeč a počítač. - Export/backup lze provést kopírováním JSON z konzole (viz tlačítko export v UI není implementováno, lze doplnit). 5) Omezení a doporučení - Není to serverová aplikace: nezabezpečené, nešifrované, není sdílené mezi zařízeními. - Pro produkci je doporučeno nasadit server (ověřování admina, e-mailová notifikace, zálohy). - Pokud chcete, mohu přidat export CSV, ověření e-mailů nebo generování PDF potvrzení.
